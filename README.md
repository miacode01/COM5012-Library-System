Here is a guide to help you navigate through this repository:

Main Page (<>Code)
There are three files: test_code, coding, my library system.
1. test_code is a folder where my first code for testing the add book feature is. 
2. coding. Originally named 'main system' this was where I showcased my changes to the code.
3. my library system.cpp -> The Smart Library Management System code is contained within this file!!

How to use the code:
1. Open Visual Studio 2022.
2. Simply download 'my library system.cpp' or copy the code over to a new c++ file.
3. Run the code!

How to use the program:

Sign Ins:

member, 2026

librarian, 2026

admin, 2026

*new user (any username or password of your choice!)

1. Enter number of the three exisiting users on the system for example: librarian, type 2.
2. Type the sign in of the user provided. (Let's use the librarian as an example!)
3. Type the number of the feature you'd like, example: 2. Add Book.
4. Write any book title you want! (Yes, you can add an extra copy of the same book! It won't affect the pre-exisitng one!)
5. You should get a message saying "Book added successfully!"

Using Keywords (Member)
1. When logged in as member, there will be an example provided when typing a keyword. If you type the letter 'F' or 'Fr' Frankenstein will automatically appear. This
   feature also works for new added books too!
